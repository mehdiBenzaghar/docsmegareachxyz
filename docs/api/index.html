API Overview

The MegaReach API provides programmatic access to our platform's features, allowing developers to integrate social automation, rewards, and analytics into their applications.

üîê Authentication

All API requests require authentication using an API key. Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Get your API key from the Dashboard ‚Üí API Keys

Base URL

https://api.megarearch.xyz/v1

Core Endpoints

GET /user/profile

Get user profile information

POST /engagement/reply

Generate AI-powered reply

GET /rewards/balance

Get $REACH token balance

POST /campaign/create

Create marketing campaign

GET /analytics/overview

Get analytics overview

Example: Generate AI Reply

Request:

POST /v1/engagement/reply Content-Type: application/json Authorization: Bearer YOUR_API_KEY { "content": "Great product! How does it work?", "context": "Product announcement tweet", "tone": "friendly", "platform": "twitter" }

Response:

{ "success": true, "data": { "reply": "Thank you for your interest! Our product uses advanced AI...", "confidence": 0.92, "tokens_earned": 5 } }

Request Parameters

Parameter Type Required Description
content string Yes The content to respond to
context string No Additional context for better responses
tone string No Response tone: friendly, professional, casual
platform string Yes Social platform: twitter, instagram, linkedin

Rate Limits

  • Free Tier: 100 requests/hour
  • Pro Tier: 1,000 requests/hour
  • Enterprise: Custom limits

Rate limit headers are included in all responses:

X-RateLimit-Limit: 1000 X-RateLimit-Remaining: 999 X-RateLimit-Reset: 1640995200

Error Handling

All errors return appropriate HTTP status codes and error messages:

{ "success": false, "error": { "code": "RATE_LIMIT_EXCEEDED", "message": "Rate limit exceeded. Try again in 60 seconds.", "details": { "limit": 100, "reset_at": "2024-01-01T12:00:00Z" } } }

SDKs & Libraries

  • JavaScript/Node.js: npm install @megarearch/sdk
  • Python: pip install megarearch
  • Go: go get github.com/megarearch/go-sdk
  • PHP: composer require megarearch/php-sdk
